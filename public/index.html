A Casual Developer

Hello! I'm Jacob, a self taught hobbyist web developer. I build things for fun as a way to express my creativity through code and to share it with the world. I got my start in coding when learning about bioinformatics and how to use R and Python to analyze massive data sets. These Days I focus on the front end of the web and love playing around with static site generators like Eleventy (Which I used to build this site) to build JAMstack sites.

Other hobbies of mine include weight lifting, cooking, planted aquariums, and DIY projects. I'll be featuring some of these on the CasualDevelopment YouTube channel to share with Y'all as breaks between coding projects.
